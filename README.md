# LetterShiftle

A small indie game made with love — created by **Skolie Dev**.  
Find me on **GitHub and X (Twitter)** as @SkolieDev.

---

## About the Game
**LetterShiftle** is a short word puzzle game where you must **unscramble the shuffled word**.  
It was developed as a personal side project to experiment with simple mechanics, logic, and a clean, relaxing design.

---

## Controls
- **Letter keys** — Type your answer  
- **Mouse** — Click the on-screen keyboard  
- **Touch screen** — Tap the on-screen keyboard (mobile)

---

## Built With
- **Engine:** Visual Studio Code  
- **Language:** JavaScript
